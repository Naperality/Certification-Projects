Introduction to HTML

Hypertext Markup Language

HTML is the standard markup language for creating and structuring web content. It provides a way to describe the structure of a document using tags and attributes. HTML documents are the building blocks of web pages.

It serves as the backbone of web content by defining the structure and elements within a document. HTML allows developers to organize and present information on the web, making it accessible to users through web browsers.

Document Structure

HTML documents follow a specific structure, consisting of essential elements. Each HTML document typically includes the following elements:

  • !DOCTYPE html

    Specifies the HTML version being used (HTML5 in most cases).

  • html

    Encloses the entire HTML document.

  • head

    Contains meta-information about the HTML document, such as title, character set, and linked stylesheets.

  • title

    Sets the title of the HTML document, displayed in the browser's title bar or tab.

  • body

    Encloses the content of the HTML document, including text, images, links, and other elements.
Text Elements

HTML provides various tags for displaying and formatting text content.

  • h1-h6

    Heading tags for defining different levels of headings.

  • p

    Defines a paragraph of text.

  • a

    Creates hyperlinks to other web pages or resources.

  • strong and em

    Emphasize text by making it bold or italic, respectively.

  • br

    Inserts a line break within text.
Structural Elements

HTML includes elements for organizing and structuring content.

  • div

    A generic container that helps organize content into sections or groups.

  • span

    A generic container for inline elements, often used for styling purposes.

  • ul, ol, and li

    Create unordered and ordered lists with list items.

  • stable, tr, th, and td

    Builds tables with rows, header cells, and data cells.
Multimedia Elements

HTML supports embedding multimedia content.

  • img

    Embeds images in the document.

  • audio and video

    Embeds audio and video files, respectively.

  • iframe

    Embeds an external webpage or content from another source.

This documentation provides a basic overview of HTML, covering essential elements for creating well-structured and formatted web documents. For more in-depth information and details, refer to the official HTML specification.
